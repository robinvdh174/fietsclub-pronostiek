# Fits Club Neet Te Snel — Pronostiek 🚴

Offline pronostiek-app van de fietsclub. Na het fietsen voorspelt
iedereen in het café de exacte uitslag van de Belgische match van de dag
(niemand mag dezelfde kiezen); op het einde van het seizoen pakt de
winnaar de pot.

**Gebruiken:** open de GitHub Pages-link van deze repo op je gsm en kies
"Zet op beginscherm" — daarna werkt de app volledig offline als gewoon
app-icoon. Alle gegevens blijven lokaal op het toestel; maak af en toe
een back-up via Instellingen → Exporteer.

De app is een dependency-vrije PWA (HTML/CSS/JS + service worker +
IndexedDB). Bron, tests en documentatie leven in de (privé) memory-vault
van de eigenaar; deze repo is enkel het publicatiekanaal.
